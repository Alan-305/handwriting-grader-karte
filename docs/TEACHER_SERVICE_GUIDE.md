# 教師向けサービス説明・利用規約（たたき台）

> **用途**: 個人教師・少人数指導の先生向け。アプリ内の初回案内、契約前の説明、利用開始時の同意文面などに転用できます。  
> **注意**: 本稿は一般的な整理に基づくたたき台です。正式提供・有料販売の前に、弁護士等の専門家による確認を推奨します。

---

## 1. サービス説明（案内用・1段落）

本サービスは、大学入試向けの個別指導を支援するため、**過去問データの取り込み・管理、AI による新規問題の生成、問題の印刷、手書き答案の読み取りと添削、添削結果に基づく解説・アドバイス**を、ひとつのワークフローで行える統合型の教育支援ツールです。入力される過去問・答案・生徒情報などのコンテンツは、利用する教師が提供するものであり、本サービスはそれらの著作権を取得するものではありません。具体的な教材の選定、取り込み、生徒への配布の方法は、利用する教師の判断と責任において行われます。

---

## 2. 推奨する使い方

本サービスは、次の使い方を推奨します。

| 用途 | 推奨 |
|------|------|
| 過去問 PDF の取り込み | 大学公式サイト等から**適法に入手した資料**を、**教師自身の分析・出題参考**として利用する |
| 生徒に配布する問題 | **AI が生成した新規問題**、または教師が作成したオリジナル問題を用いる |
| 過去問そのものの印刷・配布 | **行わない**（分析・形式把握のための内部参照にとどめる） |
| 模範解答・解説の配布 | 添削結果として確定した内容、または生成・自作した模範解答を用いる |
| 生徒データ | 指導に必要な範囲に限定し、保護者・生徒への説明責任を果たす |

## 3. コンテンツに関する注意（著作権）

利用開始前に、次の点をご確認ください。

### 3.1 取り込んでよい素材の目安

- **推奨**: 各大学・入試実施機関が公式に公開している過去問題・解答 PDF
- **推奨**: 教師自身が作成した問題・模範解答・分析メモ
- **非推奨・禁止**: 市販の問題集・赤本・黄本・予備校配布資料等を、権利者の許諾なくスキャンして取り込む行為

### 3.2 過去問データベースの位置づけ

取り込んだ過去問は、**問題形式の把握、出題傾向の分析、AI による新規問題生成の参考**として利用することを想定しています。過去問の全文を、そのまま生徒用教材として大量配布する用途は想定していません。

### 3.3 AI 生成問題について

AI が生成した問題は、必ず教師が内容を確認してから使用してください。原題と酷似する表現や構成になっていないか、生徒に配布する前にご確認ください。

### 3.4 外部 AI サービスへの送信

本サービスは、問題生成・答案読み取り・添削・アドバイス等の処理のため、入力データの一部を外部 AI サービス（例: Anthropic Claude、Google Gemini）に送信します。利用にあたり、生徒の個人情報を不必要に含めないようご配慮ください。
